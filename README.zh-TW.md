# yexca 的部落格

歡迎來到我的部落格倉庫！

這裡紀錄了我在學習、生活與開發過程中的思考、整理與創作內容。  
部落格支援多語言，目前包含簡體中文、繁體中文、日文與英文（並非每篇文章都有對應語言版本）。

## ✨ 特色

- 📚 多語言支援（Hugo + i18n）
- 🛠 自訂化主題與分類/標籤優化
- 💡 技術筆記 / 生活心得
- 🗂 架構清晰，適合快速瀏覽與二次開發

## 🗃 資料夾結構

- `content/`：文章內容，依語言分類
- `themes/`：主題採用 [stack](https://github.com/CaiJimmy/hugo-theme-stack)，並加以自訂
- `static/`：靜態資源
- `hugo.yml`：部落格設定檔

## 🧩 使用技術

- Hugo 靜態網站生成器
- GitHub Actions 自動部署
